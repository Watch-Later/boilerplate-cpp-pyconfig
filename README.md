# boilerplate-cpp-pyconfig
boiilerplate of python code as a setting file for c++

## dependencies
- VisualStudio 2015
- Boost 1.70 building with Boost.Python 64bit
- Python 3.x 64bit

## requirements
setting up the following environment variables
- BOOST_ROOT_1_70_0
- BOOST_LIB_1_70_0
- PYTHON_INCLUDE
- PYTHON_LIB
